# 目录

### cp_lookup

###### cp_trans:

```
省级行政区与地级行政区的对应关系
```

###### Inverse_dic( ):

```
反转dic对象的key和value
```

###### ptwoc( ):

```
根据省级行政区域名（及其简称）给出下辖所有地级行政区域名单
```

###### ctwop( ):

```
根据地级行政区域名（及其简称）给出所属省级行政区名
```

###### add_pro( ):

```
根据一列地级行政区名给出一列对应的省级行政区域名
```

###### fiilup_city( ):

```
对一列不完整的地级行政区名单进行名称补完
```
